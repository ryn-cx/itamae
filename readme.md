# Itamae

Crunchyroll history exporter.

## Install

[Install userscript](https://ryn-cx.github.io/itamae/index.prod.user.js) (requires [Tampermonkey](https://www.tampermonkey.net/) or similar)

## Usage

1. Go to your [Crunchyroll history](https://www.crunchyroll.com/history) page.
2. Scroll down to load all entries â€” the script captures watch-history API responses as they load.
3. Click the **Save List** button next to the page heading to download your history as JSON.
