# Itamae

Crunchyroll history exporter.

## Install

1. Install [Tampermonkey](https://www.tampermonkey.net/) or a similar userscript manager.
2. [Install Itamae](https://ryn-cx.github.io/itamae/index.prod.user.js).

## Usage

1. Go to the [Crunchyroll history](https://www.crunchyroll.com/history) page.
2. Click the **Load History** button to automatically load your watch history, or
   scroll to the bottom of the page to manually load your watch history.
3. Click the **Save History** button to download your watch history as a JSON file.
